# DIO. UNA DOMANDA

Prima bozza mobile-first del sito didattico **DIO. UNA DOMANDA**.

Stack:

- Astro per le pagine pubbliche.
- React per le interazioni dell'area classi.
- Netlify per build e pubblicazione.

## Sviluppo

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Area riservata

La prima versione usa una protezione leggera lato client per soli materiali non sensibili. Non contiene dati personali, voti, foto di minori o elaborati identificabili. Per contenuti realmente privati andrà sostituita con una protezione serverless o login dedicato.
